# Sistema de Inscripciones

Este proyecto es un formulario simple para registrar participantes y guardar los datos en Supabase.

## Despliegue en GitHub Pages

1. Sube este proyecto a GitHub.
2. En el repositorio, ve a Settings > Pages.
3. Selecciona la rama principal como fuente.
4. Guarda los cambios.
5. GitHub Pages publicará el sitio en una URL similar a:
   https://tu-usuario.github.io/tu-repositorio/

## Supabase

Asegúrate de tener:
- un proyecto creado en Supabase,
- la tabla `participantes` creada,
- y una política `INSERT` para usuarios anónimos.
